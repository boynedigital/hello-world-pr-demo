# Hello World PR Demo

A tiny practice project for learning the GitHub pull request workflow.

## What this is

This repo contains a single small script, `greet.py`, with one function:
`greet(name)`, which returns a friendly greeting string.

## Usage

```bash
python greet.py Ada
```

## Running tests

```bash
python -m unittest test_greet.py
```

Feel free to fork this repo and practice opening your own pull requests aginst it.
